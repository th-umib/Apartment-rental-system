ABOUT SMARTAPART

A smarter way to manage apartment rentals

SmartApart is an apartment rental management system built to connect users with available apartments and help admins manage listings more efficiently.

PROJECT PURPOSE

Why SmartApart was created

The purpose of SmartApart is to make apartment searching, booking, and management more organized. Instead of keeping apartment data in an unclear way, the system provides a structured platform where listings, prices, availability, bookings, and inquiries can be handled from one place.

The project includes a public side for users and an admin side for managing apartment data. Users can browse apartments, check city, address, price, and availability, while the admin can add, edit, update, and delete apartment listings.

Main idea: Public users can explore apartments and send requests, while admin users manage the system from a dedicated panel.

System Focus

  • Apartment listings
  • Booking management
  • User inquiries
  • Admin dashboard
  • Responsive design
  • Clear feedback messages
